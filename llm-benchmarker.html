ℹ️ À propos de LLM Benchmarker

LLM Benchmarker v0.01 est une maquette fonctionnelle permettant d'évaluer les performances de modèles LLM locaux et externes directement depuis le navigateur.

Runners locaux supportés :
🦙 Ollama — http://localhost:11434
🏠 LM Studio — http://localhost:1234
🦔 llama.cpp (server) — http://localhost:8080

APIs externes :
OpenAI, Mistral AI, Anthropic Claude (nécessitent une clé API)

Métriques collectées :
Tokens générés · Tokens/seconde · Temps de 1er token (TTFT) · Temps total · Température · Modèle

💾 Stockage local : localStorage — vos données restent sur votre machine.
🔑 Les clés API sont stockées localement et jamais transmises à des tiers.

🔌 Test de connectivité

Vérifiez que vos runners locaux sont accessibles.
